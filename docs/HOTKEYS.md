# HOTKEYS

`Ctrl + D` - дублируем строку

`Alt + Enter` - показать контекстные действия, например, separate lines

`Ctrl + Shift + ↑` - двигаем строку вверх, вниз

`Ctrl + B`, `Ctrl + Click` - переход к объявлению

`Ctrl + Shift + U` - toggle uppercase

`Alt + G` - выделяем следующий такой же выбранный код
