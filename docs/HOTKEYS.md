# HOTKEYS

`command + B` - проваливаемся в класс или объект или файл

`control + G` - выделяем следующий такой же выбранный код

`command + option + L` - форматирование кода

`option + Enter` - вызывает дополнительное меню для форматирования

`control + O` - вызывает дополнительное меню для переопределения методов класса

`control + I` - вызывает дополнительное меню для переопределения методов интерфейса
