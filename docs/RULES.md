# RULES

- Всегда добавляем `type` для переменной и возвращаемое значение для функции. Не используем `:=`

- Всегда указываем значение по умолчанию

- Всегда добавляем `trailing comma` для `dictionaries` и параметров `func`, чтобы при сохранении применялось `separate lines` форматирование

- Используем `snake_case` для именования переменных и функций

- Используем `CamelCase` для именования классов, `enums`
